# EasyFood

A hybrid service and e-commerce full stack application. It is a mixture between [UberEats](https://www.ubereats.com/) and [OpenTable](https://www.opentable.ca/). Built with TypeScript and Python. \

For a full documentation, refers to the `docs/` folder

## Authors

- [@thomastran117](https://www.github.com/thomastran117)
- [@sanjeeveasparan](https://www.github.com/sanjeeveasparan)

## Tech Stack

**Client:** Angular, TypeScript, TailwindCSS

**Server:** Python, FastAPI

**Database:** MongoDB, Redis

**Others:** RabbitMQ
